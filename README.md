# dotnet-mvc
Learning about MVC on dotnet

## :rocket: Technologies

- **C#**

- **.Net 5**

- **MySql**

  

## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/Maycon-Rodrigues/dotnet-mvc.git
```

#### Backend

```bash
# Go into the repository
$ cd dotnet-mvc

# Run the app
$ dotnet run
```
